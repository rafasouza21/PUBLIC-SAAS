git init . -> cria um repositório git na pasta atual

git status -> mostra o status do 
trabalho de cada arquivo 

git add nome_arquivo -> adiciona arquivo para 
ser commitado

git commit -m "msg" -> faz commit do que estava para ser commitado

git log -> mostra todas as ações commitadas

git diff -> mostra tudo oq foi alterado dentro dos arquivos 

## FLUXO DE TRABALHO
 1. criar o diretório do projeto
 2. iniciar o git dentro do diretório criado (git init.)
 3. edita ou cria arquivos
 4. git status
 5. git add <nome do arquivo>
 6. git commit -m "mensagem"
 7. volta para o passo 3
